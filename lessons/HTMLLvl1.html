HTML Level 1

CSS stands for cascading style sheet. Where HTML describes what should be in a web page, css describes how different aspects in html should be displayed

HTML uses tags to describe where your elements should appear on the web page. A few basic tags include

  • < !DOCTYPE > Tells the page what type of file the document is
  • < html > Is the root of the DOM. Contains all other elements
  • < head > Contains some metadata about the website
  • < title > Is the whats displayed on the tab at the top of the page
  • < body > Contains the main section of the page
  • < hn > Where n is a number from 1 to 6. Contains headers
  • < p > Contains text


To define a tag you need to enlcose your open tag in < tag > and to close the tag you use < /tag > (without the spaces)
Finally you enter your content in between the < tag > and the < /tag >


Basic Example:

< !DOCTYPE html >
< html >
< body >

< h1 >

< /h1 >
< p >

< /p >

< /body >
< /html >
